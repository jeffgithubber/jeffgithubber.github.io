Client : Essendon FC - Corporation

Delivery Company(s) : Friday Media | Magnum Opus

Brief Summary :
Friday Media worked in collaboration with magnum opus to create an online/viral membership drive for Essendon Football Club. The main drive of this campaign was to reach 50,000 members by creating the longest commercial with the most members involved.

Requirements :
Members were able to login via the website and upload a video in any video format accepted by YouTube. An approval system was then implemented to allow an administrator to review videos and filter them. Once approved users would then be sent an email alerting them that their video has been approved and is now displayed on the site.

Personal Involvement :

  • Project Management
  • Yoube video management integration into content management system (utilising Zend youtube gdata api)
  • Member import tool
  • Social media integration
  • Send to friend module
